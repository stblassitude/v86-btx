Bildschirmtext

Bedienung

Durch Drücken von F10 oder Klick auf die Anwahltaste ☎ wird die Verbindung zum BTX-System hergestellt.

Zum Navigieren zwischen den BTX-Seiten wird die ein- oder zweistellige Nummer hinter einem Menüeintrag eingegeben, oder per *Seitennummer# eine Seite direkt angesteuert.

Wichtig: * und # sind Sondertasten; im BTX-Decoder muss dafür F1 oder F2 gedrückt werden. Natürlich können die entsprechenden Tasten auf der Bildschirmtastatur geklickt werden.

Die Technik

Diese Seite enthält einen in JavaScript geschriebenen PC-Emulator, in dem ein historischer Bildschirmtext-Decoder (PC online 1&1 BTX-COM Version 4.34) läuft.

Über eine Websocket-Verbindung zum Server kommuniziert der Decoder mit der Emulation der BTX-Host-Software.

Diese Seite inkl. des ISO-Images, das im Emulator läuft, findet sich unter github.com/stblassitude/v86-btx.
